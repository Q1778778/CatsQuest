The following is the structure of player.json:

location: (*initial location of the player*)
  row: int
  col: int

level: int (*initial level of the player*)

strength: int (*initial strength of player*)

health: int (*initial health of player*)

qualifying experience: int (*the minimum amount of experience PER LEVEL*)

keys: string list (*keyboard buttons that control the players' movements,
e.g. ["left", "right", "up", "down"]*)
